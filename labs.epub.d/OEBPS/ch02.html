2.1. Project Atomic Lab Pre-requisites

This lab was designed so that everything can be comleted on a single laptop.

  1. Laptop Hardware

    1. Provisioned with Fedora 20 or RHEL 7
    2. Minimum 8GB RAM, 16GB preferred
    3. Minimum 50GB free space

  2. Laptop Software

    1. Fedora 20 or RHEL 6/7

    2. Virtualization software

      1. qemu-kvm
      2. virt-manager
      3. libvirt-daemon-kvm
      4. OR you can simply install the "Virtualization" group via "yum groupinstall Virtualization"
    3. The MongoDB client is required

  3. Virtual Machines

    1. One virtual machine called: Atomic_Host_1
    2. One virtual machine called: Atomic_Host_2 (Only used for the multi-host / container linking geard lab)
    3. One virtual machine called: Compose_Server
